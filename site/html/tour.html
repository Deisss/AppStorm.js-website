What is AppStorm.JS ?

  • A good start for structured app
  • RIA Application ready
  • WebServices ready (handle pretty good WebServices content)
  • Blazing fast
  • No server side
  • Company ready: easy task separation, ...

What is not AppStorm.JS

  • A CSS framework
  • A data centered framework
  • An event based framework
  • A toolkit to do everything you ever want, in 5 seconds
  • A coffee machine
